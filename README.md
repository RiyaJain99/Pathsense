## ⚡ What is PathSense?

> 🚀 AI system that predicts **pedestrian accident risk** using real-world factors like weather, road, and traffic.

🧠 Powered by Machine Learning  
📍 Live location + smart dashboard  
🎤 Voice-enabled accessibility  

---

## 🎯 System Flow



📍 Location + Inputs
↓
🧠 ML Model (XGBoost)
↓
⚠️ Risk Prediction
↓
📊 Dashboard + Voice Output


---

## ✨ Features

- 📊 Real-time risk prediction  
- 🗺 Live map & location tracking  
- 🎤 Voice input + audio output  
- ⚡ Fast FastAPI backend  
- 📈 ML-powered insights

## 🚀 Run Locally

### Backend
```bash
cd api
uvicorn main:app --reload --port 8000

👉 http://localhost:8000

Frontend
cd frontend
python -m http.server 5500

👉 http://localhost:5500

🧠 Models Used
XGBoost (main)
Logistic Regression
Random Forest
Decision Tree
📁 Structure
pathsense/
├── api/
├── frontend/
├── ml/
├── dataset/
```
⚠️ Note
Uses real-world features like weather & traffic
Best run on localhost for maps + permissions
